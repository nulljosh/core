Q: What's jot?
A: jot is a programming language with classes, functions, JSON, and HTTP. Here's an example:

fn greet(name) {
    return "Hello " + name + "!";
}
print greet("Joshua");

Q: Can you write a function?
A: Sure! Here's a jot function:

fn fibonacci(n) {
    if (n <= 1) {
        return n;
    }
    return fibonacci(n - 1) + fibonacci(n - 2);
}

Q: Show me a class.
A: Here's a jot class:

class Person {
    fn init(name, age) {
        this.name = name;
        this.age = age;
    }
    fn greet() {
        return "Hi, I'm " + this.name;
    }
}

let p = new Person("Joshua", 26);
print p.greet();

Q: What about error handling?
A: jot has try/catch:

try {
    let data = parse(json_str);
    print data;
} catch (e) {
    print "Error: " + e;
}

Q: Can you do HTTP requests?
A: Yes! Here's how:

Q: Show me functional programming.
